🔢 线性回归 & 逻辑回归

从“预测连续值”到“二分类决策”，掌握最基础也最重要的两类监督学习模型。

一、线性回归（Linear Regression）

目标：预测一个连续值（如房价、温度、销量）。

模型形式

y = w1*x1 + w2*x2 + ... + b

损失函数

常用均方误差 MSE：

MSE = (1/n) * Σ(y_pred - y_true)^2

优化方法

  • 正规方程（小数据可用）
  • 梯度下降（更通用）
适合线性关系明显的数据；如果关系非线性，需要特征工程或换模型。

二、逻辑回归（Logistic Regression）

目标：做分类（最常见是二分类，如“是/否”、“好/坏”）。

核心思想

先做线性组合，再经过 Sigmoid 映射到 0~1 概率：

p = 1 / (1 + e^(-z)), z = w*x + b

损失函数

常用对数损失（交叉熵 Binary Cross-Entropy）。

分类规则

通常用阈值 0.5：p >= 0.5 判为正类，否则负类。

虽然叫“回归”，但逻辑回归本质上是分类模型。

三、两者对比

  • 输出：线性回归输出连续值；逻辑回归输出概率/类别
  • 任务：线性回归用于回归；逻辑回归用于分类
  • 损失：MSE vs 交叉熵

四、最小 Python 示例（scikit-learn）

1) 线性回归

from sklearn.linear_model import LinearRegression

X = [[1], [2], [3], [4]]
y = [2, 4, 6, 8]

model = LinearRegression()
model.fit(X, y)
print(model.coef_, model.intercept_)
print(model.predict([[5]]))

2) 逻辑回归

from sklearn.linear_model import LogisticRegression

X = [[0], [1], [2], [3]]
y = [0, 0, 1, 1]

clf = LogisticRegression()
clf.fit(X, y)
print(clf.predict([[1.5], [2.5]]))
print(clf.predict_proba([[1.5], [2.5]]))
